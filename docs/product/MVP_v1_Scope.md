# MVP v1 Scope

## In Scope

- controlled commercial pre-bid demo flow
- tender summary, requirements, participant requirements, risk, and decision artifacts
- bounded LLM-assisted drafting/analysis with schema validation
- operator review actions and audit trail
- TKP collection workspace with manual registration
- deterministic economics and bid-readiness checks
- customer-facing demo report artifacts
- pilot package docs for a controlled commercial MVP

## Explicit Boundaries

- metadata slices from `M-049` and `M-050` remain bounded to internal registry/library/control usage
- human review is required before any final decision state
- outputs may recommend and structure, but may not execute external actions
