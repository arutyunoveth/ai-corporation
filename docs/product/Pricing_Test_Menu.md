# Pricing Test Menu

## Purpose

This menu is for pilot-stage pricing conversations only. It is not billing automation and it is not a public pricing page.

## Test Shapes

### Option A: Single Controlled Scenario

- one operator-guided pilot scenario
- fixed setup and review fee
- best for first validation meeting

### Option B: Small Tender Wave

- limited batch of controlled pilot tenders
- includes operator workflow rehearsal and evidence review
- best for customers comparing multiple opportunities

### Option C: Controlled LLM Add-On

- optional provider-backed analysis under schema validation and human review
- priced separately because it adds provider cost and review overhead

## Explicit Exclusions

- no submission handling fee because submission is not performed by the repository
- no supplier outreach automation fee because automation is out of scope
- no production deployment fee because SaaS hardening is not included
