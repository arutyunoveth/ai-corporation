# Pilot Offer Description

## Offer Shape

The controlled commercial pilot is a bounded operator-assisted engagement for pre-bid analysis and bid-preparation support.

## Included

- one approved pilot scenario or customer-like tender package at a time
- pre-bid report generation
- requirements and risk review
- manual TKP/economics/readiness workspace
- controlled operator workflow and evidence capture
- review of limitations and next-step fit

## Excluded

- autonomous submission
- procurement platform integration
- EDS/signature work
- supplier messaging automation
- legal guarantee of bid success
- post-award execution

## Delivery Mode

- manual-control
- human-review-required
- internal operator present for every gating action
- controlled LLM mode optional, schema-validated, and review-bound

## Pilot Exit Question

After the pilot, the customer should be able to answer:

- was the report useful?
- did the workflow reduce manual pre-bid effort?
- were the limitations acceptable?
- is a broader paid pilot justified?
