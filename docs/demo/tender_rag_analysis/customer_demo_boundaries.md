# Границы клиентского демо

## Что показываем

- локальный анализ закупки;
- структуру отчёта;
- progress по секциям;
- источники;
- историю запусков;
- экспорт DOCX/PDF.

## Что не показываем как готовый production

- многопользовательскую авторизацию;
- production queue;
- SLA;
- автоматическую подачу заявки;
- юридически значимое решение без проверки человеком;
- универсальную поддержку всех типов закупок;
- OCR всех сканов;
- интеграции с любыми ЭТП из коробки.

## Как правильно формулировать

- "Помогает ускорить первичный анализ."
- "Формирует черновик аналитического отчёта."
- "Сохраняет источники для проверки."
- "Подходит для пилота на ограниченном наборе сценариев."

## Как не формулировать

- "Полностью заменяет тендерного специалиста."
- "Гарантирует решение участвовать или не участвовать."
- "Автоматически подаст заявку на любую площадку."
- "Уже production-ready SaaS."
