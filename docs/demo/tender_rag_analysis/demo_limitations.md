# Ограничения демо

## 1. Это demo/MVP, не production

- Синхронная подготовка — нет очереди задач
- Нет фонового job manager
- Нет полноценного multi-user режима
- Один backend-процесс, один поток

## 2. Ограничения документов

- OCR не реализован (скан-образы PDF не обрабатываются)
- Часть PDF может быть пустой, если это скан
- DOC/RTF/ZIP/RAR extraction в backlog
- Качество анализа зависит от извлечённого текста

## 3. Ограничения LLM

- Ответ не является юридическим заключением
- Отчёт требует проверки тендерным специалистом
- Модель может ошибаться — важны citations
- LLM работает только на русском языке
- Контекстное окно 4096 токенов

## 4. Ограничения интеграции

- Демо работает на подготовленном Mac mini runtime
- Для нового заказчика потребуется настройка источников данных
- First-time подготовка закупки может занимать время (загрузка с ЕИС, извлечение текста)
- LLM-генерация отчёта занимает ~2.5 минуты для 10 разделов

## 5. Ограничения EIS / внешний доступ

- Доступность ЕИС может влиять на загрузку данных
- Сетевые настройки / proxy важны
- Live network tests вынесены из default pytest

## 6. Что уже хорошо

- Полностью локальный контур
- Повторяемый анализ (idempotent prepare)
- Источники / citations в отчёте
- REST API для интеграции
- Demo UI для ручного прогона
- Офлайн-тесты (252 теста tender_research, 1074+ всего)
