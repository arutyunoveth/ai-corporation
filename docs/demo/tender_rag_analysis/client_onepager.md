# AI-агент Арвектум для первичного анализа закупки

## Какую проблему решает

- много документов;
- много ручного анализа;
- сложно быстро понять риски и требования;
- нужен отчёт для обсуждения внутри команды.

## Что делает агент

- анализирует документы закупки;
- выделяет ключевые разделы;
- показывает источники;
- формирует отчёт;
- экспортирует результат в DOCX/PDF.

## Что получает клиент

- ускорение первичного разбора;
- единый формат отчёта;
- история запусков;
- проверяемые источники;
- локальный контур.

## Что нужно для пилота

- 5–10 типовых закупок;
- примеры документов;
- критерии качества;
- эксперт для проверки результатов;
- 1–2 недели на настройку сценария.

## Важное ограничение

- агент не заменяет юридическую или тендерную экспертизу;
- итоговое решение остаётся за человеком.
