# Broader Internal Usage Review Cadence

## Wave-Level Cadence

- setup review after `B1-S1`
- review after Wave #1
- stability review after Wave #2
- final exit review in `B1-S4`

## Within-Wave Cadence

For each active deal in a wave, review at:

1. wave entry
2. screening / qualification
3. supplier / commercial decision
4. finance / risk checkpoint
5. bid / procedure checkpoint
6. payment / claim checkpoint if reached
7. closure / learning checkpoint

## Helper Cadence

- `/events?deal_id=...` at every critical transition
- `/dashboards/build` at major stage changes
- `/workspace-feed/build` before operational handoffs
- `/action-queue/build` when approvals matter
- `/launch-visibility/build` once per active review window

- Master Plan matched: yes
