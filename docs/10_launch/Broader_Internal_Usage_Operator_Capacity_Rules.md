# Broader Internal Usage Operator Capacity Rules

## Purpose

Define the maximum safe operator load for broader internal usage.

## Capacity Rules

1. Wave scope must stay within a consciously supervised operator pool.
2. No wave may expand beyond what the assigned operators can review manually.
3. Helper rebuild cadence must remain realistic for the chosen deal count.
4. If manual review begins slipping, the wave must pause rather than silently normalize overload.

## Required Capacity Checks

- number of active deals per operator
- number of critical review points expected in the wave
- ability to rebuild and inspect helper visibility surfaces on time
- explicit owner/reviewer availability

## Stop Threshold

If the operator pool cannot sustain the planned cadence without skipping manual review, the wave must pause.

- Master Plan matched: yes
- What changed vs plan: capacity limits expressed as explicit operating rules instead of raw numeric quotas
- Any drift introduced: `NO`
