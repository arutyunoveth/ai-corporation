# CEO Instruction — Dry Run 0

## Task

Assess whether Arvectum is ready for the first paid restricted tender-operations pilot.

## Context

Arvectum is building a modular AI-assisted procurement and tender-operations system.

The current product has:
- tender intake;
- RFQ-first workflow;
- supplier questions;
- TKP comparison;
- economics;
- calibrated contract risk;
- bid/no-bid recommendation;
- human-control boundaries;
- restricted pilot docs;
- company operations agents metadata;
- Hermes-compatible manual context exports.

## Required assessment

Evaluate readiness from the perspective of:

1. Tender operations.
2. Finance and unit economics.
3. Legal and contract risk.
4. QA and release readiness.
5. CEO decision requirements.

## Output required

Prepare a CEO Decision Memo with one of the following recommendations:

- GO_TO_FIRST_PAID_RESTRICTED_PILOT
- GO_WITH_FIXES
- NO_GO_BLOCKED

## Hard boundaries

Do not recommend autonomous execution.
Do not recommend cloud dispatch for confidential data.
Do not recommend external communication without CEO approval.
Do not assume legal or financial obligations can be accepted automatically.
